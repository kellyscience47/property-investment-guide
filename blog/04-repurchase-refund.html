什麼是重購退稅？

重購退稅是政府為了鼓勵換屋而設的優惠：賣掉自住房後買新房（或先買新房再賣舊房），已繳的房地合一稅可以申請退還。

這是除了自住優惠之外，另一個超級重要的節稅工具。兩者甚至可以搭配使用——先用自住優惠降低稅率，再用重購退稅把剩餘稅金退回來。

重購退稅的基本條件

不論「先買後賣」還是「先賣後買」，都必須符合以下條件：

✅ 必要條件

  1. 自住用途：出售的舊屋和購入的新屋都必須是自住用途
  2. 設籍要求：本人、配偶或未成年子女在舊屋和新屋都有設籍
  3. 無出租營業：兩間房屋都不能有出租或營業使用
  4. 時間限制：買賣之間不超過2年
  5. 完成設籍：新屋必須在買入後完成戶籍遷入

⏰ 時間計算

  • 先賣後買：出售舊屋完成移轉登記日起算2年內，購入新屋並完成移轉登記
  • 先買後賣：購入新屋完成移轉登記日起算2年內，出售舊屋並完成移轉登記

先買後賣 vs 先賣後買

先賣後買

舊屋出售 → 繳稅 → 2年內買新屋 → 申請退稅
  • 先賣再買，稅是先繳後退
  • 需要先準備一筆稅金
  • 退稅金額看新舊屋價差

先買後賣

買新屋 → 2年內賣舊屋 → 申報時直接扣抵（可能不用先繳）
  • 先買再賣，在申報舊屋所得時可直接扣抵
  • 現金壓力較小
  • 但需要同時有兩間房的資金

退多少？大換小 vs 小換大

小換大（新屋價格 ≥ 舊屋價格）

已繳稅額全額退還！ 🎉

項目 金額
舊屋售價 2,000萬
舊屋房地合一稅 80萬
新屋購價 2,500萬
退稅金額 80萬（全額）

大換小（新屋價格 < 舊屋價格）

按比例退稅。公式：

退稅金額 = 已繳稅額 ×（新屋價格 ÷ 舊屋價格）
項目 金額
舊屋售價 2,500萬
舊屋房地合一稅 100萬
新屋購價 1,800萬
退稅比例 1,800萬 ÷ 2,500萬 = 72%
退稅金額 100萬 × 72% = 72萬

完整試算範例

範例：自住優惠 + 重購退稅雙重節稅

情境： 王先生自住滿6年的房子賣出，同時買了新房。

項目 舊屋 新屋
價格 出售 3,000萬 購入 3,500萬
原始購入價 2,000萬 —
成本費用 150萬 —

Step 1：計算課稅所得 - 3,000萬 - 2,000萬 - 150萬 = 850萬

Step 2：適用自住優惠 - 400萬免稅 - (850萬 - 400萬) × 10% = 45萬稅金

Step 3：適用重購退稅 - 新屋 3,500萬 > 舊屋 3,000萬 → 小換大 → 全額退稅 - 退稅金額：45萬

最終結果：繳稅 0 元！ 🎊

若不使用任何優惠（持有5年以上一般稅率）： - 850萬 × 20% = 170萬稅金

雙重優惠省下 170萬。

重購退稅的5年追蹤期

⚠️ 重要！退稅後有5年限制：

申請退稅後的5年內，新屋不得： - 改為出租使用 - 供營業使用 - 遷出戶籍（本人、配偶、未成年子女都遷出） - 再次出售

若違反以上任一條件，退稅金額將被追繳（含利息）。

例外情況

  • 未成年子女成年後遷出：不影響（只要本人或配偶仍設籍）
  • 因工作或就學短期遷出：需個案認定，建議保留證明
  • 新屋因災害毀損：不追繳

申請流程

先賣後買（事後申請退稅）

  1. 出售舊屋後30日內申報並繳納房地合一稅
  2. 2年內購入新屋，完成移轉登記
  3. 向新屋所在地國稅局申請退稅
  4. 檢附文件：
    • 重購退稅申請書
    • 新舊屋買賣契約書
    • 舊屋房地合一稅繳款書
    • 戶籍謄本
    • 新屋所有權狀
  5. 審核通過後約1-2個月退稅入帳

先買後賣（申報時直接扣抵）

  1. 購入新屋，完成設籍
  2. 2年內出售舊屋
  3. 申報舊屋房地合一稅時，同時申請重購扣抵
  4. 經核准後，可少繳或免繳稅金

常見問題 Q&A

Q1：重購退稅和自住優惠可以同時使用嗎？

可以！ 這是最強的組合。先適用自住優惠（400萬免稅＋10%稅率），繳了稅之後再申請重購退稅把稅金退回來。

Q2：夫妻各有一間自住房，可以互相重購退稅嗎？

可以。例如先生名下的舊屋賣掉，買入太太名下的新屋，只要符合自住和設籍條件就能申請。

Q3：新屋可以登記在配偶名下嗎？

可以。重購退稅允許新屋登記在本人或配偶名下。

Q4：預售屋可以當作「新屋」嗎？

要等交屋後才算。 重購退稅的時間是看移轉登記日，預售屋簽約日不算。所以要注意：從舊屋出售到新屋交屋過戶，不能超過2年。

Q5：我賣了房但還沒決定買哪裡，可以先繳稅之後再退嗎？

可以。先賣後買的模式就是先繳稅，只要在2年內買到新的自住房，就能申請退稅。
